

# 脑肿瘤智能诊断分析系统

## 简介

脑肿瘤智能诊断分析系统是一个基于深度学习的医学影像分析平台，提供脑肿瘤的分类、检测、分割和视频诊断功能。系统采用Flask框架构建Web应用，集成了多种AI模型，为医疗机构和医生提供智能辅助诊断工具。
演示视频链接🔗：https://www.bilibili.com/video/BV1RiL8zPEyd/?share_source=copy_web&vd_source=2da3835cf91a5e486f5b560ecb77f289

## 功能特性

### 用户认证模块
- 用户注册：支持自定义头像选择（多种医生头像可选）
- 用户登录：安全的身份验证机制
- 用户退出：清除会话信息
- 个人资料管理：支持修改基本信息和密码

### 影像诊断模块
- 脑肿瘤分类：使用分类模型识别肿瘤类型
- 肿瘤检测：定位图像中的肿瘤区域
- 结果展示：显示诊断结果、置信度和建议
- 图像对比：原始图像与诊断结果对比展示

### 图像分割模块
- 肿瘤分割：精确分割脑肿瘤区域
- 参数调节：支持调整高斯模糊、对比度和阈值参数
- 交互式预览：实时查看分割效果
- 分割结果保存：导出分割后的图像

### 视频诊断模块
- 实时视频流处理
- 视频帧逐帧分析
- 视频控制：开始、暂停、停止检测
- 检测结果记录和导出

### 病历记录管理
- 病历记录：保存诊断记录
- 记录查询：支持搜索和筛选
- 记录详情：查看完整诊断信息
- 批量操作：批量删除记录
- PDF导出：生成诊断报告PDF

### 数据分析模块
- 数据统计：展示诊断数据统计信息
- 图表可视化：使用图表展示诊断趋势
- 数据导出：支持数据导出功能

## 技术栈

- **后端框架**：Flask
- **深度学习框架**：PyTorch
- **数据库**：SQLite (users.db)
- **前端技术**：HTML5, CSS3, JavaScript
- **图表库**：Chart.js
- **AI模型**：
  - classification.pt：分类模型
  - detection.pt：检测模型
  - segmentation.pt：分割模型

## 安装部署

### 环境要求
- Python 3.7+
- CUDA（推荐，用于GPU加速）

### 安装步骤

1. 克隆项目到本地：
```bash
git clone https://gitee.com/dpdd01/brain-tumor-system.git
cd brain-tumor-system
```

2. 创建虚拟环境：
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```

4. 启动应用：
```bash
python app.py
```

5. 访问系统：
打开浏览器访问 http://localhost:5000

## 使用指南

### 用户注册与登录
1. 访问系统首页，点击注册按钮
2. 选择喜欢的医生头像
3. 填写用户名、密码等信息完成注册
4. 使用注册的账号登录系统

### 影像诊断
1. 登录后进入诊断页面
2. 点击上传区域选择脑部影像图片
3. 系统自动进行分析，显示诊断结果
4. 查看分类结果、置信度和建议信息

### 图像分割
1. 进入分割页面
2. 上传原始医学影像
3. 使用滑动条调节参数：
   - 阈值（Threshold）：调整分割阈值
   - 高斯模糊（Gaussian Blur）：降噪处理
   - 对比度（Contrast）：调整图像对比度
4. 实时查看分割效果，满意后保存结果

### 视频诊断
1. 进入视频诊断页面
2. 点击开始按钮启动摄像头
3. 系统实时分析视频流中的每一帧
4. 暂停或停止检测，查看检测结果
5. 导出检测报告

### 病历管理
1. 在导航栏点击"记录"进入病历页面
2. 使用搜索框查找特定病历
3. 点击记录查看详细信息
4. 可进行编辑、删除等操作
5. 支持批量选择和删除

### 数据分析
1. 进入数据分析页面查看统计图表
2. 查看诊断数据分布和趋势
3. 支持数据刷新和导出

## 注意事项

1. 本系统仅供辅助诊断使用，最终诊断结果应由专业医生确认
2. 确保上传的医学影像符合系统要求（支持格式：PNG, JPG, JPEG）
3. 建议在GPU环境下运行以获得更好的性能
4. 请妥善保管账户密码，不要泄露给他人
